About Historical Weather Explorer

Exploring past weather patterns through interactive data visualization

Project Overview

Historical Weather Explorer is a web-based application designed to help users explore and visualize historical weather data for locations around the world. By combining a searchable location interface with interactive charts, the project allows users to better understand how weather conditions have changed over time.

This project was developed as part of a university course focused on web technologies, data systems, and information architecture.

What the Application Does

  • Allows users to search for a location using city names or coordinates
  • Fetches historical weather data from a public weather API
  • Displays temperature and weather trends using visual charts
  • Helps users identify long-term patterns and anomalies

Data Source

Weather data is retrieved from Open-Meteo's Historical Weather API. The API provides archived historical weather observations such as temperature, precipitation, and weather conditions for specified locations and date ranges.

This project does not store personal user data. All data retrieval is performed on demand when a user submits a search.

Technologies Used

  • HTML5 – Semantic structure and content layout
  • CSS3 – Styling, layout, and responsive design
  • JavaScript – Frontend logic, API requests, and dynamic UI updates
  • Node.js – Backend runtime environment
  • Express.js – Server-side API routing
  • Supabase – Database for storing user search history
  • Chart.js – Interactive data visualization
  • Leaflet – Interactive map rendering
  • Open-Meteo APIs – Geocoding and historical weather data
  • Vercel – Application deployment and hosting
